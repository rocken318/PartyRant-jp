## 概要

PartyRant-jp の家族・学習向けバージョン。小学生〜高校生が学年・教科を選んでクイズを楽しめるアプリ。家族みんなで遊ぶホストモードと、子供がひとりで練習できるソロモードの両方を提供する。

---

## アーキテクチャ

### 基本方針
- PartyRant-jp をフォークして新リポジトリ `PartyRant-fam` を作成
- 技術スタック: Next.js (App Router) + Supabase + TypeScript（jp と同一）
- Supabase プロジェクトは新規作成（jp とは DB を分離）
- リポジトリは完全独立（jp とは別々に進化させる）

### jp から削除するもの
- `opinion` ゲームタイプ（trivia + polling のみ残す）
- 多言語対応 i18n（日本語のみに簡素化）

### jp から追加・変更するもの
- 問題メタデータ: `grade`（1〜12）・`subject`
- 学年×教科フィルターロジック
- ソロプレイモード（ログイン不要）
- PartyRant-Family 専用ランディングページ

## データモデル

### Question の追加フィールド

```ts
interface Question {
  // 既存フィールド（id, order, text, imageUrl, options, correctIndex, timeLimitSec）はそのまま
  grade: number;              // 1=小1, 2=小2, ..., 6=小6, 7=中1, 8=中2, 9=中3, 10=高1, 11=高2, 12=高3
  subject: Subject;           // 下記 enum
  gameType: 'trivia' | 'polling';  // 問題ごとに設定
}

type Subject =
  | 'japanese'   // 国語
  | 'math'       // 算数/数学
  | 'science'    // 理科
  | 'social'     // 社会
  | 'english'    // 英語
  | 'ethics'     // 道徳
  // 後から文字列を追加するだけで拡張可能
```

### 学年グループの表示名

| grade 値 | 表示 |
|---------|------|
| 1〜6 | 小1〜小6 |
| 7〜9 | 中1〜中3 |
| 10〜12 | 高1〜高3 |

### 問題バンク（Supabase）
- 問題は `questions` テーブルに grade + subject 付きで格納
- ランダム出題クエリ: `grade BETWEEN gradeMin AND gradeMax AND subject = selectedSubject`
- ホスト作成ゲームのカスタム問題は jp と同様の構造

## 認証

- **プリセット問題でのプレイ**: ログイン不要（現行 jp と同じ）
- **カスタム問題の作成・管理**: メール認証必要（現行 jp と同じ）
- **子供ソロモード**: ログイン不要で即スタート

## UI フロー

### トップ画面（ログイン不要）

```
[ PartyRant Family ]
  ┌─────────────────┐
  │  🎮 みんなで遊ぶ  │  ← ホストモード（ルーム作成）
  └─────────────────┘
  ┌─────────────────┐
  │  📚 ひとりで練習  │  ← ソロモード（即スタート）
  └─────────────────┘
  ┌─────────────────┐
  │  🔑 コードで入室  │  ← 参加者（現行と同じ）
  └─────────────────┘
```

### ソロモード フロー

1. 学年範囲を選ぶ（例: 小3〜小5）
2. 教科を選ぶ（タブ形式）
3. 問題数を選ぶ（5 / 10 / 20問）
4. 即スタート
5. 結果表示（正解率表示、ランキングなし）

### ホストモード フロー

- jp と同じ（イベント → ゲーム作成 → 参加者待ち → 進行）
- ゲーム作成時に「プリセット問題から選ぶ」で学年×教科フィルターを追加
- カスタム問題作成もそのまま利用可能

## ビジュアル方針

- PartyRant のネオポップ骨格（太ボーダー・影）を継承
- カラー: ピンクベースからグリーン×イエロー×ブルーのファミリーカラーに変更
- フォント・角丸を少し大きめにして子供にやさしく
- ゲームタイプは trivia（正解あり）と polling（道徳等）の2種類のみ

---

## スコープ外（今回は含まない）

- opinion ゲームタイプ
- 多言語対応
- 保護者向けダッシュボード（進捗管理等）
- 問題の難易度フラグ（grade で代替）
