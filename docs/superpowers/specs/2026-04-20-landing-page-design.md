# PartyRant-jp マーケティング LP 設計書

## 概要

PartyRant-jp のマーケティング用ランディングページ（以下 LP）を `/lp` ルートに新設する。現行の `/` ページはそのまま維持する。

**Goal:** 飲み会・企業イベント両方のユーザーを獲得し、「今すぐ試したい」と思わせるLP

**Architecture:** Next.js App Router の Server Component として `/app/lp/page.tsx` に実装。i18n は next-intl（`messages/ja.json` に `lp` 名前空間追加）。既存のデザインシステム（neobrutalism、pr-pink、Bebas Neue + DM Sans）を踏襲しつつ、ヒーロー部分はグラデーション背景でマーケティングLPとしての迫力を出す。

**Tech Stack:** Next.js 15 App Router、next-intl、Tailwind CSS、既存デザイントークン（pr-pink / pr-dark / Bebas Neue / DM Sans）

## ページ構成（上から順）

### 1. ヒーローセクション

- 背景: pr-pink グラデーション（`from-[#FF0080] to-[#FF4DAA]`）、ボトムに大きな波形クリップパス
- キャッチコピー（Bebas Neue）:
  - メイン: 「宴会でも、リリースイベントでも。」
  - サブ: 「リアルタイムで盛り上がれるクイズ&投票アプリ」
- スマホモックアップ画像（SVGで代替可能 — 投票画面のネオブルータリズム風イラスト）
- CTA ボタン: 「無料ではじめる」→ `/auth/login`、「プリセットを試す」→ `/presets`

### 2. ユースケースセクション

3つのシーンカード（横並びスクロール or グリッド）:

| シーン | 絵文字 | 見出し | 説明 |
|--------|--------|--------|------|
| 飲み会 | 🍺 | 飲み会クイズで盛り上がれ | 幹事ゼロ準備、参加者スマホで即スタート |
| 企業イベント | 🏢 | リリース発表をインタラクティブに | 来場者全員がリアルタイム参加 |
| 社内研修 | 📚 | 研修を双方向にアップデート | アンケート結果がその場で可視化 |

### 3. 機能一覧セクション

見出し: 「3つのゲームモード」

- **クイズ（トリビア）**: 正解を競う。ポイント制で盛り上がり保証
- **アンケート（ポーリング）**: その場で意見を集計。棒グラフでリアルタイム表示
- **少数派バトル**: 多数派 vs 少数派で笑いが生まれる

「✨ AI クイズ自動生成（New）」: トピックを入れるだけで問題を自動生成。準備時間ゼロ。バッジ: `NEW` ピンクラベル

追加機能:
- コード参加（アプリ不要）
- 最大参加人数無制限
- 結果をリアルタイム集計・表示

### 4. 使い方ステップセクション

見出し: 「3ステップではじめる」

```
① ゲームを作成 → ② コードをシェア → ③ みんなで盛り上がる
```

各ステップにアイコン + 説明テキスト。ネオブルータリズムの番号バッジ。

### 5. FAQ セクション

アコーディオン形式（`<details>/<summary>` HTML要素）:

1. 参加者はアプリのインストールが必要ですか？ → 不要、ブラウザだけでOK
2. 何人まで参加できますか？ → 制限なし
3. 問題は自分で作れますか？ → ホスト登録で作成可能、AIで自動生成も可能
4. 無料で使えますか？ → プリセットは完全無料、アカウント登録で作成機能利用可

### 6. 最終 CTA セクション

- 背景: pr-dark
- 見出し: 「次のパーティー、PartyRantで盛り上げよう。」
- ボタン: 「無料ではじめる」（pr-pink）、「プリセットを試す」（白）

## デザイン方針

- **ベース**: 既存ネオブルータリズム（太いボーダー、シャドウ、角ばったボタン）
- **拡張**: ヒーローのみグラデーション背景 + 大きめのビジュアルでインパクト
- **フォント**: Bebas Neue（見出し）、DM Sans（本文）
- **カラー**: pr-pink (#FF0080)、pr-dark (#111)、白
- **レイアウト**: `max-w-[480px] mx-auto` モバイルファースト（PCでも480pxセンタリング）
- **アニメーション**: 最小限（Tailwind の transition のみ）
